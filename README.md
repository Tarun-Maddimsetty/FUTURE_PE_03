# FUTURE_PE_03
# AI SEO Blog & Content Cluster Generator for Cafe

## 📌 Project Overview
This project is part of the Prompt Engineering Internship task.

The objective of this task is to use AI tools to generate SEO optimized blog content for a business website.

SEO (Search Engine Optimization) helps websites rank higher in Google search results and attract more visitors.

In this project, we created SEO blog content for a Cafe business using structured prompts in ChatGPT.

The generated content includes:
- Blog outline (H1, H2, H3)
- Full SEO blog article
- Content cluster ideas
- Internal linking ideas
- Prompts used to generate content

This project demonstrates how AI can be used to create professional SEO content for real-world business needs.

## ✍ Prompts Used

### Prompt 1 – Blog Outline

Create SEO optimized blog outline for topic:

Benefits of Drinking Coffee

Include:
H1 title
H2 headings
H3 subheadings

Make outline SEO friendly and simple.

---

### Prompt 2 – Full Blog Article

Write SEO optimized blog article for:

Benefits of Drinking Coffee

Word limit: 600-800 words
Target audience: coffee lovers
Tone: simple and informative

Include headings and easy language.

---

### Prompt 3 – Content Cluster Ideas

Give 8 SEO blog topic ideas related to coffee business.
Make topics suitable for cafe website blog.

---

### Prompt 4 – Internal Linking Ideas

Give internal linking ideas for coffee blog topics.
Explain how blogs can connect to each other.
